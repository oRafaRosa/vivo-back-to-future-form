# Diretrizes Visuais

## Conceito
O visual deve misturar:

- Vivo 5G
- De Volta Para o Futuro
- Inteligência Artificial
- Tecnologia corporativa
- Futurismo neon

## Paleta sugerida

- Preto profundo: `#030008`
- Grafite escuro: `#0B0614`
- Roxo Vivo/neon: `#6600CC`
- Roxo claro neon: `#9B2DFF`
- Lilás brilho: `#C084FC`
- Branco: `#FFFFFF`
- Cinza texto: `#B8B8C7`

## Elementos visuais obrigatórios

- Vivinho/logo Vivo 5G
- Carro estilo De Volta Para o Futuro
- Fundo escuro
- Roxo neon
- Elementos de IA/tecnologia
- Linhas de tempo, partículas, grid ou circuitos

## Layout

- Card central com glassmorphism escuro.
- Bordas roxas suaves.
- Glow discreto nos elementos principais.
- Header com logo Vivo 5G/Vivinho.
- Hero inicial com carro e rastro neon.
- Barra de progresso em roxo neon.

## Animações sugeridas

- Fundo com grid se movendo lentamente.
- Partículas roxas discretas.
- Glow pulsante no logo ou no botão principal.
- Entrada dos cards com fade + leve deslocamento vertical.
- Transição entre etapas com Framer Motion.
- Rastro neon atrás do carro.

## Cuidado
Não transformar o formulário em uma peça muito pesada. O visual precisa ser impactante, mas o preenchimento deve continuar simples.
